キューブには、接続が切れたり電源を切ってから入れても設定が維持される「保存可能な設定」があります。
これらの設定は[設定](configuration.md)と同じ特性（characteristic）を使って変更や初期化ができます。

| プロパティ          | 値                                                             |
| ------------------- | -------------------------------------------------------------- |
| Characteristic UUID | 10B201FF-5B3B-4571-9508-CF3EFCD7BBAE                           |
| Properties          | [Write](#書き込み操作), [Read](#読み出し操作), [Notify](#通知) |
| Descriptor          | Configuration                                                  |

## 書き込み操作

### スピーカー消音の設定

**機能ボタンを押した状態** で、以下のデータを書き込むことでスピーカー消音を設定できます。
結果の取得方法は[スピーカー消音の設定の応答](#スピーカー消音の設定の応答)を参照してください。

| データ位置 | タイプ | 内容       | 例                                                        |
| ---------- | ------ | ---------- | --------------------------------------------------------- |
| 0          | UInt8  | 設定の種類 | <span class="fixed">`0x33`</span>（スピーカー消音の設定） |
| 1          | UInt8  | Reserved   | `0x00`                                                    |
| 2          | UInt8  | 消音設定   | `0x01`                                                    |

| 消音設定 | 意味                                                          |
| -------- | ------------------------------------------------------------- |
| `0x00`   | 消音設定を解除（初期値）                                      |
| `0x01`   | すべての音を消音                                              |
| `0x02`   | システム音（起動・終了、接続・切断などの音）のみ消音 |

消音設定を受け付けた場合は、ランプの色（消灯・白色・消灯）と音（消音を設定した場合は鳴らない）で通知します。

### 保存した設定の初期化

**機能ボタンを押した状態** で、以下のデータを書き込むことでキューブ内に保存した**すべて**の設定を初期化できます。
結果の取得方法は[保存した設定の初期化の応答](#保存した設定の初期化の応答)を参照してください。

| データ位置 | タイプ | 内容       | 例                                                            |
| ---------- | ------ | ---------- | ------------------------------------------------------------- |
| 0          | UInt8  | 設定の種類 | <span class="fixed">`0x0f`</span>（保存した設定の初期化） |

初期化を受け付けた場合は、ランプの色（消灯・白色・消灯）と音で通知します。

## 読み出し操作

### スピーカー消音の設定の応答

[スピーカー消音の設定](#スピーカー消音の設定)を書き込むと以下の応答を行います。

| データ位置 | タイプ | 内容       | 例                                                              |
| ---------- | ------ | ---------- | --------------------------------------------------------------- |
| 0          | UInt8  | 設定の種類 | <span class="fixed">`0xb3`</span>（スピーカー消音の設定の応答） |
| 1          | UInt8  | Reserved   | `0x00`                                                          |
| 2          | UInt8  | 結果の値   | `0x00`（設定成功）                                              |

結果の値の意味は以下のとおりです。

| 結果の値    | 意味            |
| ----------- | --------------- |
| `0x00`      | OK・設定成功    |
| `0x00` 以外 | NG・設定失敗 ※1 |

※1 機能ボタンを押していない場合は要求を受け付けない。（受け付け時の演出なし。）

### 保存した設定の初期化の応答

[保存した設定の初期化](#保存した設定の初期化)を書き込むと結果に応じて以下の応答を行います。

| データ位置 | タイプ | 内容       | 例                                                                  |
| ---------- | ------ | ---------- | ------------------------------------------------------------------- |
| 0          | UInt8  | 設定の種類 | <span class="fixed">`0x8f`</span>（保存した設定の初期化の応答） |
| 1          | UInt8  | Reserved   | `0x00`                                                              |
| 2          | UInt8  | 結果の値   | `0x00`（初期化成功）                                                |

結果の値の意味は以下のとおりです。

| 結果の値    | 意味              |
| ----------- | ----------------- |
| `0x00`      | OK・初期化成功    |
| `0x00` 以外 | NG・初期化失敗 ※1 |

※1 機能ボタンを押していない場合は要求を受け付けない。（受け付け時の演出なし。）
